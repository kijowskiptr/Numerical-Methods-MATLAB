# Numerical Methods & Signal Modeling in MATLAB

Ten projekt zawiera implementacje kluczowych algorytmów numerycznych opracowanych w ramach moich studiów na AGH oraz mojej własnej ciekawości.

## 1. Linear Systems Solver
Narzędzie do rozwiązywania układów równań liniowych ($Ax = b$).
- **Metody bezpośrednie:** Eliminacja Gaussa, Rozkład LU (dekompozycja).
- **Metody iteracyjne:** Gauss-Seidel (z analizą zbieżności).
- **Analiza:** Porównanie czasu wykonywania i stabilności numerycznej dla macierzy różnych rozmiarów.

## 2. Data Approximation & Modeling Tool
Narzędzie do rekonstrukcji i modelowania sygnałów.
- **Interpolacja Lagrange'a:** Idealna rekonstrukcja sygnału (z prezentacją efektu Rungego).
- **Metoda Najmniejszych Kwadratów (LSM):** Aproksymacja wielomianowa (trendu) dla danych zaszumionych.
- **Metryki jakości:** Implementacja RMSE, MAE oraz współczynnika determinacji $R^2$.

## Jak uruchomić?
Otwórz dowolny plik `main.m` w środowisku MATLAB i naciśnij F5.
